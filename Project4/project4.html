Overview

NAVER CONNECT을 React를 이용하여 Main 페이지와 Sub(Research) 페이지 퍼블리싱

#React #Swiepr #toggle #Route

Problems & Solution

1. React

React 배움이 적음으로 다양한 문제들 발생 React를 배움이 적어 style 적용 부분 등 문제 발생 시 문제가 원인이 무엇인지 확인하고 구글을 통해서 문제를 해결함.

2. Swiper

슬라이더가 작동하지 않는 문제 발생 Swiper.js에 구글에서 다양한 Swiper 적용 방식을 확인하며 직접 적용을 해보면서 확인 중 SwiperCore와 Pagination 등을 이용하여 문제를 해결함 슬라이더가 작동은 되었지만 각 페이지 영역의 너비가 달라지는 문제가 발생. 슬라이더 각 영역에 최대, 최소의 너비값을 주어 문제를 해결함.

3. toggle

toggle이 반응 하지 않는 문제가 발생 HTML의 방식대로 작동이 되지 않아 구글을 통해 다양한 방법들을 확인하면서 직접 적용 시켜보며 확인 중 useState와 on,off 함수 구현해 문제를 해결함

4. Route

Route 첫 사용으로 미숙한 부분의 의한 적용이 되지 않는 문제가 발생 Route의 홈페이지에서 공부를 하여 components와 BrowserRouter, Routes, Link 등 사용방법을 배워 적용시켜 문제를 해결함 Route와 Link로 페이지 넘어가는 것은 적용이 되었지만 path="/"로 준 Route가 보이지 않는 문제가 발생 최근에 Route가 바뀐 것에 구글을 통해서 정보를 얻는 것에 어려움을 겪었지만 다양한 방법을 적용하며 확인 중 BrowserRouter를 HashRouter로 바꾸니 적용이 되어 문제를 해결함.

Review

react를 이용하여 첫 작업이었는데 react를 아직 배우는 단계 중이여서 그런지 다양한 문제가 발생을 하게 되었는데 문제의 원인을 확인부터 해결 하는 부분까지 다양한 어려움으로 인해 너무 막막했다. 하지만 어렵다고 포기하지 않고 구글을 통해서 공부를 하면서 원인이 무엇인지 확인을 하고 그 원인을 해결 할 방법이 무엇인지 하나씩 직접 적용을 해보면서 문제를 해결해 나가고자 생각을 했다. swiper, toggle, Route 등 react를 통해서 이번에는 적용을 시켜보았지만, 아직 공부가 부족하여 복잡하고 어렵다고 느껴진다. 이번에 사용한 swiper, toggle, Route 뿐만 아니라 다양한 것을 적용시킬 수 있도록 공부를 해야겠다고 생각을 하게 되었다.
